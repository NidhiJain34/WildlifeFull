Wildlife Sanctuary Management System
Description

This project is a Java-based application designed to manage wildlife sanctuary data efficiently. It allows users to manage animals, staff, medical records, and generate reports using a menu-driven interface.

Features
 Animal Management (Add, Update, Delete, View)
 Medical Records Management
 Staff Management (Vets & Rangers)
 Sanctuary Information
 Reports & Analytics

 Technologies Used
Java
JDBC (Java Database Connectivity)
MySQL
VS Code

Note-
This project is developed as part of academic coursework to demonstrate OOP concepts and database integration.
